# Kalkan Development Phases

1. Project foundation: Compose, Material 3, Hilt, navigation, theme.
2. Authentication: Google Sign-In, guest mode, user profile, role checks.
3. Home: status card, safe/help/location/SOS actions, recent earthquakes, emergency contacts.
4. Earthquakes: AFAD service, list, filters, nearby earthquakes, detail screen.
5. Emergency: SOS, safe notice, help request, location sharing, emergency contacts.
6. Offline: Room-backed contacts, disaster bag, first aid, emergency phones, assembly areas.
7. Notifications: FCM topics, notification center, city-based alerts.
8. Admin: announcements, emergency alerts, user stats, notification management, settings.
9. Quality: accessibility, error states, loading states, restrained transitions.
10. Release: security rules, Crashlytics, Analytics, Play Store assets, privacy policy, closed test.
